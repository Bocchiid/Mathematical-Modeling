## 项目概述

本分析采用**双路径回归模型**，对比研究影响评委评分和粉丝投票的因素差异。通过建立两个独立的OLS回归模型，分别解释：

- **评委路径（Judge Path）**：评委评分的影响因素
- **粉丝路径（Fan Path）**：粉丝投票意向的影响因素

这种对比分析帮助我们理解：
- 评委和粉丝重视的因素是否存在偏差
- 各个特征对两条路径的相对重要性
- 模型预测效能的差异

| Judge_Score | 评委评分（标准化后） | 官方评分数据 |
| Vote_Share | 粉丝投票占比（标准化后） | 投票统计数据 |

### 特征工程（Feature Engineering）

#### 1. **高斯年龄隶属度 (Age_Gauss)**
```
Age_Gauss = exp(-(age - 30)? / (2 × 10?))
```
- 中心值：30岁
- 标准差：10岁
- **解释**：在特定年龄范围内的表演者可能受偏好，用高斯函数模拟这种非线性偏好

#### 2. **职业舞者实力 (Pro_Power)**
```
Pro_Power = 1 / (平均排名)
```
- **计算方法**：按舞蹈搭档分组，用排名倒数作为实力指标
- **解释**：与优秀职业舞者搭档的明星表现更佳

#### 3. **地理因素二分类 (Is_US)**
```
Is_US = 1 if 明星来自美国 else 0
```
- **解释**：美国本土明星可能在投票中获得优势

#### 4. **行业分组 (Industry_Group)**
- **方法**：选取top 5行业类别，其余归为"Other"
- **包含行业**：Entertainment, Sports, Politics等
- **解释**：不同背景的明星在评委/粉丝中的吸引力差异

### 数据处理
- **缺失值处理**：删除关键变量缺失的行
- **标准化处理**：使用StandardScaler对所有特征进行z-score标准化
  - 使得系数直接可比
  - 便于解释标准化回归系数

## 分析方法

### 双路径OLS回归模型

#### 模型1：评委路径
```
Judge_Score = β? + β?×Age_Gauss + β?×Pro_Power + β?×Is_US 
             + β?×Industry_1 + β?×Industry_2 + ... + ε?
```

#### 模型2：粉丝路径
```
Vote_Share = γ? + γ?×Age_Gauss + γ?×Pro_Power + γ?×Is_US 
            + γ?×Industry_1 + γ?×Industry_2 + ... + ε?
```

### 方法论优势
? **直接对比**：同样的特征在两条路径上的效应大小  
? **系数可解释性**：标准化系数表示标准差变化带来的标准差变化  
? **统计显著性**：每个系数都附带p值，用于假设检验  
? **模型诊断**：残差分析、Q-Q图、拟合优度评估

## 核心发现

### 1. 两条路径存在显著差异
- 评委和粉丝重视的因素在**方向或大小**上存在明显差异
- 这反映了不同评估主体的价值判断差异

### 2. 关键影响因素排序
基于绝对系数值的特征重要性排名揭示了：
- 最具影响力的3-5个因素
- 可忽略的边际因素（系数接近零）

### 3. 年龄与职业搭档效应
- **年龄**：通过高斯隶属度捕捉非线性偏好
- **职业舞者**：实力强的搭档带来显著加成

### 4. 地理与行业因素
- **美国本土因素** (Is_US)：在粉丝投票中可能更重要
- **行业背景**：不同行业明星获得的支持差异显著

## 建议与展望

### 短期建议（模型改进）

1. **检查多重共线性**
   - 计算方差膨胀因子 (VIF)
   - 若VIF > 10，考虑删除或合并特征

2. **异常值检测**
   - 使用Cook距离识别有影响的观测值
   - 考虑Robust回归处理离群值

3. **交互项探索**
   - 测试年龄×职业搭档的交互效应
   - 添加地理×行业的交互项

4. **非线性关系**
   - 尝试二次项或其他多项式
   - 考虑分段线性回归

### 中期建议（业务应用）

5. **路径差异解释**
   - 深入分析为何两条路径在某些特征上相反
   - 是否反映评委/粉丝的不同价值取向

6. **预测模型构建**
   - 使用该模型预测新明星的评分/投票
   - 评估预测误差和置信度

7. **特征子集优化**
   - 用AIC/BIC选择最优特征组合
   - 平衡模型复杂度和解释力

### 长期建议（战略洞察）

8. **时间序列扩展**
   - 纳入季度、赛季等时间维度
   - 分析评委/粉丝偏好的演变趋势

9. **因果推断强化**
   - 当前为相关分析，需实验设计验证因果
   - 考虑工具变量法或倾向得分匹配

10. **模型组合**
    - 结合多个模型（回归+树模型+神经网络）
    - 使用投票法或Stacking提升预测性能
